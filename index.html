blog386

A Node.js Blogging Platform, using the 386-Bootstrap theme.

This platform will let you make your presence known painfully on the web by reducing the server to the speed of a late 80's 32-bit CPU

Painfully bring your clients to their knees with this awesome recreation of old DOS software in EGA mode

Installation

Like all good DOS Software this is a pain to install, setup your postgres db - create a database, and run the schema.sql script - once that is done be sure to load the data in blog_data.sql. Otherwise the program will just crash and burn.

Once that is done the login is admin with the password test, you might want to change it quickly, before you are raped on the internet
